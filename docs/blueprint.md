# **App Name**: Apex Residences

## Core Features:

- Property Listings: Display a grid of real estate property listings with key details and images sourced from Firestore.
- Advanced Search and Filters: Implement search filters to find properties by type, location, price range, and other criteria.
- Interactive Property Maps: Show property locations on an interactive map, with the ability to filter based on location.
- User Authentication: Allow users to create accounts, sign in, and save favorite properties.
- Contact Form with Notifications: Enable users to submit inquiries through a contact form, triggering email notifications via Firebase Functions.
- AI Property Description Generator: Generate a unique and engaging property description tool, which incorporates the square footage, number of bedrooms, style and neighborhood features to engage a prospective buyer. The generated description is presented to the admin.
- Content Management System: Admin portal using Firebase authentication, functions, and Firestore rules to upload images of property and agent information.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for trust and professionalism.
- Background color: Light gray (#F9FAFB) for a clean and modern look.
- Accent color: Gold (#D4AF37) to highlight key elements.
- Headline font: 'Playfair', a modern sans-serif with an elegant, high-end feel, perfect for headings and short amounts of text. For longer body copy use 'PT Sans', which matches the styling but is better for long runs of text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use sharp, minimalist icons for property features and categories.
- Implement a clean and structured layout inspired by leading real estate websites, emphasizing clear navigation and easy access to property information.
- Use subtle animations to highlight content and interactive elements.
